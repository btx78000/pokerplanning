🔴 ÉTAPE 1: ARRÊT COMPLET
═══════════════════════════════════════════════════════════════

Double-cliquez sur le fichier:

    ►  clean-restart.bat  ◄

Ce fichier va:
✓ Tuer TOUS les processus Node.js
✓ Nettoyer les ports 3001 et 5173
✓ Redémarrer le backend avec les nouvelles config CORS
✓ Redémarrer le frontend avec les nouvelles config Socket.io

⏱️  ATTENDEZ 10 SECONDES après le lancement!


═══════════════════════════════════════════════════════════════

🟢 ÉTAPE 2: VÉRIFICATION NAVIGATEUR
═══════════════════════════════════════════════════════════════

1. Ouvrez votre navigateur

2. Allez sur:  http://localhost:5173

3. Appuyez sur:  Ctrl + Shift + R
   (Hard refresh pour vider le cache)

4. Ouvrez la Console (F12)

5. Vous devriez voir:
   ✅ Connected to server
   [vite] connected.

6. Le point de statut devrait être VERT
